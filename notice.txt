Sunrise Capture v3.6.8 안내

- 회사 PC와 V3/EDR 환경에서 정상 업무용 프로그램임을 설명하기 쉽도록 보안팀 제출용 신뢰 패키지를 자동 생성합니다.
- SecurityTrustPackage 폴더에는 프로그램 개요, 자체 서명 인증서 정보, 설치파일/실행파일 SHA256, 네트워크 통신 URL, 설치 경로, 동작 설명, 업데이트 정책, 릴리즈 이력이 정리됩니다.
- PyInstaller 실행파일 버전 리소스와 Inno Setup 설치파일 메타데이터를 보강했습니다.
- 기존 v3.5.7의 UI 브랜드명 잘림 수정, v3.5.6의 자체 코드서명, v3.5.5의 이메일 문의, v3.5.4의 다중 모니터 플로우 액션바, v3.5.3의 공지 변경 알림 기능은 유지합니다.

주의: 자체 서명은 신뢰 등록이 없는 PC에서 자동으로 공인 신뢰를 얻는 방식은 아닙니다. 회사 PC에서는 보안팀이 공개 인증서를 신뢰 목록 또는 허용 정책에 등록해야 효과가 있습니다.

이 파일은 latest.json을 수정하지 않고도 공지사항 본문만 바꾸고 싶을 때 사용할 수 있습니다.
GitHub Pages 저장소에 notice.txt 또는 notice.md로 올려두면 앱 공지사항에 반영되고, 내용 변경 시 1회 알림이 표시됩니다.
